Cash-Flow-Advanced Salary & Expense Tracker<br>
A sleek, real-time financial dashboard built to demonstrate core JavaScript proficiency, DOM manipulation, and integration with third-party libraries and APIs.

1.Real-time Calculations: Automatically updates remaining balance and budget status as you add or remove expenses.

2.Data Persistence: Integrated localStorage to ensure financial data is saved across browser sessions.

3.Dynamic Visualization: A live Pie Chart (via Chart.js) comparing total expenses against the remaining budget.

4.Currency Conversion: Real-time exchange rate integration (INR to USD) using the ExchangeRate-API.

5.PDF Reporting: Generate and download a professional summary of expenses using jsPDF.

6.Budget Alerts: Visual warnings (UI color shifts and animations) when the balance drops below 10% of the total salary.
